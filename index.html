Women Driven By Auto aims to provide women the opportunity to develop a meaningful career path in the specialized, rapidly evolving automotive industry. Members Only Forum

This members only app is an online forum where only members can create new posts/view post details and only admins can delete posts. Users can sign up for an account, log in and out, and update account details such as first name, last name, membership status, admin status, and password. To change membership/admin status and to delete posts, users must correctly input the member password (Member2024$) and admin password (Admin2024$). The frontend of this application is built with EJS, JavaScript, and Tailwind CSS, and the backend is built with Node.js, Express, and PostgreSQL. Authentication is implemented using Passport.js and secured by bcryptjs. All forms are validated with express-validator.

Knights Travails

Knights Travails is an app intended to calculate and display the minimum number of moves and route a chess piece would need to get from a start point to an end point on a chess board. This app uses a custom graph like data structure and breadth first search to calculate the knights moves. This project was built using HTML, JavaScript, CSS, and Webpack for bundling the modularized codebase.
